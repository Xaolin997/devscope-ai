# Roadmap

## Concluído

- Estrutura monorepo
- Backend Fastify
- PostgreSQL e Prisma
- Cadastro e login
- Perfil autenticado
- CRUD de empresas
- CRUD de projetos
- Permissões básicas
- Erros de domínio
- Testes de integração
- Rate limiting
- Swagger/OpenAPI

## Próximas etapas

### Backend

- Sprints
- Tarefas
- Comentários
- Convites
- Papéis mais granulares
- Paginação e filtros
- Auditoria
- Notificações
- Uploads
- Refresh tokens

### Frontend

- Autenticação
- Dashboard
- Empresas
- Projetos
- Sprints e tarefas
- Estados de carregamento e erro
- Integração segura com a API

### Produto

- Métricas de projeto
- Relatórios
- Automação de fluxo
- Recursos assistidos por IA
